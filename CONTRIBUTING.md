# Contributing to MassFinder

Thank you for your interest in contributing to MassFinder — a free, open-source tool helping Catholics find Mass, Confession, and Adoration across New England.

## Getting Started

- Read `CLAUDE.md` for agent context and code conventions
- Read `docs/ANTI_PATTERNS.md` before writing any code
- Read `docs/FORK_GUIDE.md` if adapting MassFinder for a new region

## Pull Request Process

1. Fork the repo and create a feature branch
2. Follow all conventions in `CLAUDE.md` (CommonJS, no arrow functions, no optional chaining)
3. Test on mobile Chrome and Safari before submitting
4. PRs are reviewed in batches — expect 2–5 day turnaround

## Code of Conduct

This is a mission-driven Catholic project. Contributions should serve the app's users: parishioners trying to find sacraments. Keep that north star in every decision.
