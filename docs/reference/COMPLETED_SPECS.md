## Naming Convention

Spec IDs use a 3-letter prefix + 2-digit number:
- **SFD** = Saved / Find / Detail
- **EMT** = Events / More Tab
- **CDC** = Church Detail Card
- **PTR** = Prayer Tools Reader

Future specs should continue this pattern with new prefixes.
